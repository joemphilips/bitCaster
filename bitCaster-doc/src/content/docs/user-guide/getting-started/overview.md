# Overview

bitCaster is a prediction market platform that uses Cashu ecash tokens for private, peer-to-peer trading on event outcomes. By combining Conditional Timed Fungible (CTF) tokens with Nostr-based oracle announcements, bitCaster enables trustless, censorship-resistant markets without requiring user accounts or identity verification.
