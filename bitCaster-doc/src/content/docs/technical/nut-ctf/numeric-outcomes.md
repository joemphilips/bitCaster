# Numeric Outcomes

Numeric outcome markets allow trading on events that resolve to a number within a defined range. CTF tokens for numeric markets use specialized keyset structures to represent positions across outcome intervals.

For the full specification, see the [CTF Numeric spec on GitHub](https://github.com/joemphilips/bitCaster/blob/main/nuts/CTF-numeric.md).
